访问地址拼接规则

[基础结构] https://api.iconify.design/{prefix}:{icon-name}.{format}?{parameters}

部分说明
api.iconify.designIconify 官方公共 API 域名
{prefix}图标集前缀（如 cil 表示 CoreUI Icons）
{icon-name}图标名称（如 dog）
{format}输出格式（通常为 svg、json 等）
?{parameters}可选的查询参数（如颜色、尺寸等）

常用查询参数

参数示例值作用
colorred
%2300ff00		图标颜色（支持 HEX/RGB/关键词）
%23为#的转义
width / height24
1.5rem		尺寸（像素或 CSS 单位）
fliphorizontal水平/垂直翻转
rotate90旋转角度（90°/180°/270°）
downloadtrue强制下载而非显示

颜色值可以是：

  • HEX 格式（需转义 # → %23）：?color=%236a86c8
  • 无前缀 HEX 格式：?color=6a86c8
  • 颜色名称：?color=blue
  • RGB/A 格式：?color=rgb(106,134,200)
