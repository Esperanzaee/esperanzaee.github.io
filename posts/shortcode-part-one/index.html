对话框

参考：眠于水月间 - Hugo | 在 Stack 主题上可行的短代码们.

先来展示一下:

Alice 10:30 AM Hello, how are you today?
You 10:32 AM I'm good, thanks!
Bob 10:35 AM Check out these screenshots!

对话框对应的短代码，放置在layout/shortcodes/message.html下：

# Ingredient Calculator
<link rel="stylesheet" href="/css/message.css" />

<div class="message {{if eq (.Get "from") "self"}}--self{{end}}">
    <div class="message__inner">
        {{if ne (.Get "accountID") ""}}
        <img class="avatar" src="https://unavatar.io/{{.Get "accountID"}}" width="24" height="24">
        {{else}}
        <img class="avatar" src="" width="24" height="24" style="visibility:hidden;">
        {{end}}
        <div class="message__text">
            <hstack class="message__meta">{{.Get "name"}} <spacer></spacer> {{.Get "timestamp"}}</hstack>
            {{.Inner}}

            {{if ne (.Get "images") ""}}
            <div class="grid-container">
                {{range split (.Get "images") ","}}
                <div class="grid-item">
                    <img class="message__img" src="{{.}}" alt="Image">
                </div>
                {{end}}
            </div>
            {{end}}
        </div>
    </div>
</div>

前后篇导航

参考：椒盐豆豉 - 博客前后篇导航.
预览图

对应的代码, 放置在layouts/partials/post-nav.html：

{{ $current := . }}

<!-- 获取相邻文章：按发布日期排序 -->
{{ $pages := where .Site.RegularPages "Section" "==" .Section }}
{{ $prev := $pages.Next . }}
{{ $next := $pages.Prev . }}

{{ if or $prev $next }}
<div id="post-hr"></div>
<div class="post-nav">
  {{ with $prev }}
    <a href="{{ .Permalink }}" title="上一篇: {{ .Title }}"><< 前 | {{ .Title | truncate 15 "..." }}</a>
  {{ end }}

  <a onclick="goToRandomPost()" style="cursor: pointer" title="随机阅读一篇文章">
    随便逛逛
  </a>

  {{ with $next }}
    <a href="{{ .Permalink }}" title="下一篇: {{ .Title }}">{{ .Title | truncate 15 "..." }} ｜ 后 >></a>
  {{ end }}
</div>
<div id="post-hr"></div>
{{ end }}
